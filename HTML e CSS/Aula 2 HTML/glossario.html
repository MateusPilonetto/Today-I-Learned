Glossário

Tag H1 (Heading 1)

Define o título principal da página. É o texto mais importante para os motores de busca (como o Google) e geralmente é o maior texto visível. Deve haver apenas um por página.

Tag P (Paragraph)

Define um parágrafo de texto. O navegador adiciona automaticamente um espaço (margem) antes e depois dele para separar os blocos de leitura.

Tag BR (Break)

Realiza uma quebra de linha forçada dentro de um texto. Diferente do P, ele não cria um novo bloco com espaçamento, apenas "pula" para a linha de baixo (como um "Enter" no Word).

Tag HR (Horizontal Rule)

Cria uma linha horizontal visual que atravessa a página. É usada semanticamente para separar seções de conteúdo ou mudar de assunto.

Tag UL (Unordered List)

Cria uma lista não ordenada, ou seja, aquela com "bolinhas" (bullet points). Usada para itens onde a ordem não importa (ex: lista de compras).

Tag OL (Ordered List)

Cria uma lista ordenada, ou seja, numerada (1, 2, 3...). Usada para tutoriais ou passos onde a sequência é obrigatória.

Tag A (Anchor)

Cria um link (âncora) que leva o usuário para outra página, site ou parte do mesmo documento.

Tag IMG (Image)

Insere uma imagem na página. Ela não tem fechamento (img) e precisa do atributo src para saber qual arquivo carregar.
