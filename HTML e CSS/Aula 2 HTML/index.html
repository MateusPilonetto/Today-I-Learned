Desbloquear o bootloader Xiaomi

  1. Você precisa ter a permissão para desbloqueio no app Xiaomi Community.
  2. Com a permissão, abra o miUnlock no seu computador
  3. Conecte seu smartphone Xiaomi no modo bootloader
  4. Clique em unlock.
  5. Se houver algum tempo de espera, volte no programa após este tempo e tente de novo.
